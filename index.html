01 アカウントの初期設定を行う

このセクションで達成すること

アカウントの初期設定を完了し、二段階認証を設定する

1. サインインする

1. ブラウザで[https://swipetalk.melbridge.com]にアクセスします。

2. [サインイン]画面で アカウント名を入力し[次へ]を押します。

⚠ 注意事項・補足事項

アカウント名および仮パスワードは、MelBridge事務局からあなた宛てにメールでお送りしています。

件名：しゃべり描き翻訳_アカウント発行のご連絡

3. [パスワード]画面で 仮パスワードを入力し[サインイン]を押します

4. [パスワードの更新] 画面が表示されます。現在の仮パスワードと新しいパスワードを入力し、 [サインイン] を押します。

⚠ 注意事項・補足事項

アカウント名・仮パスワードはMelBridge事務局からあなた宛てにメールでお送りしています。

ワンポイントアドバイス

パスワードは8文字以上で、大文字・小文字・数字・記号を含める必要があります。忘れないようにメモしておきましょう。

2. Authenticator(二段階認証アプリ)を設定する

1. Microsoft Authenticatorアプリに関する案内が表示されます。
[QRコードをスキャンします]の画面が表示されるまで[次へ]を押します。

2. 手元にスマートフォンを用意します。

3. スマートフォンで【Microsoft Authenticator】アプリを起動します。

4. [QRアイコン] または [Verified IDs]を押し、QRをスキャンする画面に移動します

5. PCのブラウザに表示されたQRコードを、スマートフォンで読み取ります。

⚠ 注意事項・補足事項

アプリをお持ちでない場合は、お使いのスマートフォンのアプリストアからMicrosoft Authenticatorを入手してください

スマートフォンのカメラアプリ/コードスキャンアプリからQRコードを読み取っても認証されません。必ず【Microsoft Authenticator】アプリを使用してください。

3. Authenticatorで認証する

1. PCのブラウザの[次へ]を押します。次の画面で2桁の番号が表示されます。

2. PCのブラウザに表示された2桁の番号を、スマートフォンのアプリ画面で入力し[Yes]を押します。

3. 認証が完了します。スマートフォンのアプリ画面に、登録したアカウントが表示されたことを確認します。

4. PCで[通知が承認されました]を確認し[次へ]→[完了]を押します。

5. ログインが完了します。

⚠ 注意事項・補足事項

[サインインの状態を維持しますか？] で「今後このメッセージを表示しない」をチェックすると次回以降が楽です。

03 基本の操作を行う

このセクションで達成すること

アプリの基本的な操作方法を習得する

1. サインイン

1. ブラウザで https://swipetalk.melbridge.com にアクセスします。

2. [サインイン]画面で アカウントを選択すると、アプリ画面が開きます。

⚠ 注意事項・補足事項

パスワードを求められたら、あなたが設定したパスワードを入力してください。パスワードを設定した覚えがない場合は、初期設定 1の3以降の手順を実行したか確認してください。

Microsoft Authenticatorの認証を求められたら、初期設定 3の1~5の手順を行ってください。

2. 新しいシートを作成し、シートタイトルを付ける

1. [シートボタン]を押します

2. [シートの新規作成]で新しいシートが作成されます。過去のシートはリストで保持されます。

3. シートタイトルを押すと任意の名称に変更できます

⚠ 注意事項・補足事項

シートは10枚まで保持可能。

3. 使用したい言語を選択する

1. しゃべり描きマイクの上にある[言語選択ボタン]を押します。

2. 使用したい言語を選択し[決定]を押します。

3. しゃべり描きマイクと言語が連動して設定されます。

⚠ 注意事項・補足事項

利用可能な言語は「製品仕様-翻訳言語」を参照。

4. しゃべり描きを使う

1. 左右いずれかのしゃべり描きマイクボタンを押します。

2. しゃべり描きマイクボタンが青色になったら、音声を入力します。

⚠ 注意事項・補足事項

マイクボタンを押してから音声入力できるようになるまで短いラグがあります。必ずボタンが青くなるのを確認してから音声入力してください。

3. キャンバスの任意の場所をマウス・指でなぞると、文字が表示されます。

⚠ 注意事項・補足事項

ボタンが青くなってから話し始めてください。途中キャンセルは再押下またはパレット操作で可能。

5. トランスクリプトを使う

1. 左右いずれかのトランスクリプトマイクボタンを押します。

2. ボタンが青色になったら音声入力します。

3. 入力は吹き出し単位で区切られ、翻訳/折返し翻訳が表示されます。

4. 終了するには有効なボタンを再度押してグレー表示を確認します。

5. 片側が有効中にもう一方を押すと認識言語が切り替わります。

⚠ 注意事項・補足事項

自動終了はしません。10分連続で有効の場合は強制終了します。

6. 写真・画像を貼り付ける

A. 写真を撮影して貼り付ける

A-1. ツールパレット内の[カメラボタン]を押します

A-2. 撮影画面で[カメラボタン]を押し撮影

A-3. [これを使用]で中央に配置

B. 端末に保存された画像を貼り付ける

B-1. [画像ボタン]を押してギャラリーから選択

C. クリップボードの画像を貼り付ける

C-1. 画像コピー後、[カーソル]選択で Ctrl+V で貼り付け

7. お絵描きを行う

1. [ペンボタン]でキャンバスをなぞる

2. 太さは3種類から選択可能

3. 色は[●]から選択

⚠ 注意事項・補足事項

2指タッチ描画は非対応

8. 要素を移動する

1. [カーソル]を選択

2. 対象を押してドラッグ

3. 空き領域を押して確定

⚠ 注意事項・補足事項

重なりは最前要素が選択されます

9. 要素を消去する

A. オブジェクト消しゴム

1. [消しゴム]選択後、なぞって消去

B. Delete キー

1. [カーソル]選択

2. 対象を選び Delete

10. しゃべり描きの文字を編集する

1. [カーソル]を選択

2. 一度押して枠表示→もう一度押して編集

3. 編集後[決定]で反映（翻訳/折返しも更新）

11. キャンバスエリアを画像として書き出す

1. [書き出し]を押す

2. ブラウザのダウンロードで保存

⚠ 注意事項・補足事項

保存先はブラウザ設定に依存。書き出し範囲は表示中のズーム/ウィンドウサイズに依存します。
